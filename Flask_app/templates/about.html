About

This Catalogue was developed as part of the Databases and Information-Systems course 2024 at the University of Copenhagen.

It is a simple web application that allows users to search and filter through a list of Bachelor/Masters theses. The projects are stored in a postgresql database and are displayed on the website using Flask.

Users can search for a project by typing in the search bar and clicking the search button. They can also filter the projects by selecting a level from the dropdown menu and clicking the filter button.
